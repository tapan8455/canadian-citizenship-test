## 🚀 Quick Start Guide

### **1. Install Dependencies**
```bash
npm install
```

### **2. Setup Environment**
Create `.env.local`:
```env
NEXTAUTH_URL=http://localhost:3000
NEXTAUTH_SECRET=your-secret-key
DATABASE_URL=file:./data/citizenship-test.db
```

### **3. Initialize Database**
```bash
npm run db:setup
```

### **4. Start Development**
```bash
npm run dev
```

### **5. Deploy**
```bash
# For Windows
scripts/deploy.bat

# For Mac/Linux
./scripts/deploy.sh
```
